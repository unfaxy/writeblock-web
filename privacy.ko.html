개인정보처리방침

최종 업데이트: 2026년 6월 16일

요약

일기는 기기 안에 머뭅니다 — iCloud 동기화를 켜면 당신만의 iCloud에도 저장되며, 이건 우리도 볼 수 없습니다. 우리는 자체 서버가 없고, 우리가 받는 데이터는 익명 사용 통계뿐 — 당신의 글·사진·음성은 절대 받지 않습니다. 설정에서 언제든 끌 수 있습니다.

개요

Writeblock("우리", "당사" 또는 "앱")은 일기를 쓸 때까지 선택한 앱을 차단해 습관을 만드는 일기 앱입니다. 이 개인정보처리방침은 당사가 당신의 정보를 어떻게 다루는지 설명합니다.

요점은 이렇습니다. 당사는 자체 서버도, 사용자 계정도 없습니다. 일기는 당신의 기기 안에 있고, iCloud 동기화를 켜면 Apple이 관리하는 당신 개인 iCloud 계정에도 저장되며, 이건 당사가 볼 수 없습니다. 기기를 벗어나 당사로 가는 데이터는 익명의 집계 사용 통계뿐이며 — 이마저도 끌 수 있습니다.

Writeblock이 저장하는 데이터

Writeblock은 아래 데이터를 기기 안에만 저장합니다.

  • 작성한 일기 (텍스트 내용, 날짜, 시각)
  • 앱 설정 (차단 시작 시간, 알림 설정, 테마 선택)
  • 스트릭 데이터 (현재 스트릭, 최장 스트릭)
  • 차단 앱 선택 (Apple의 Screen Time API가 관리하는 불투명 토큰)
  • 앱 잠금 PIN (iOS 키체인에 안전하게 저장되며 기기를 절대 벗어나지 않음)

데이터 수집

Writeblock은 신원을 식별할 수 있는 정보를 수집하지 않습니다. 구체적으로는,

  • 광고 식별자 없음
  • 기기 핑거프린팅 없음
  • 앱 간·사이트 간 추적 없음
  • 외부 서비스로의 크래시 리포트 없음
  • 사용자 계정이나 회원가입 없음
  • 일기 내용·사진·음성은 기기를 절대 벗어나지 않음

익명 사용 통계

사람들이 앱을 어떻게 쓰고 어디서 막히는지 이해하기 위해, Writeblock은 익명의 집계 퍼널 이벤트("온보딩 시작", "첫 일기 저장", "구매 완료" 등)를 PostHog로 전송합니다. 이 이벤트는,

  • 일기 내용과 신원 식별 정보를 전혀 포함하지 않습니다
  • 익명 식별자를 사용합니다 — 개인 프로필을 만들지 않으며, 이벤트를 당신의 신원과 연결할 수 없습니다
  • SDK가 자동으로 붙이는, 신원을 식별하지 않는 기술적 맥락(OS 버전, 앱 버전, 기기 종류, SDK 버전)만 포함합니다
  • 세션 리플레이와 자동 수집(autocapture)을 사용하지 않습니다
  • 서버 단에서 IP 기반 위치 추정을 사용하지 않습니다
  • EU에 호스팅됩니다
  • 설정 → "익명 사용 통계 보내기"에서 언제든 끌 수 있습니다

서드파티 서비스

Writeblock은 위에서 설명한 익명 사용 통계를 위해 단 하나의 서드파티 SDK — PostHog(EU 호스팅, 설정에서 옵트아웃 가능) — 를 사용합니다. 그 외 모든 기능은 Apple의 네이티브 프레임워크로 만들어졌습니다.

  • PostHog — 익명 사용 통계 전용 (일기 내용·개인정보 없음)
  • SwiftUI — 사용자 인터페이스
  • Core Data — 로컬 저장
  • Screen Time API / FamilyControls — 앱 차단
  • StoreKit 2 — App Store 구매 검증, Apple이 처리
  • CloudKit — 선택적 iCloud 동기화, Apple이 관리
  • LocalAuthentication — Face ID / Touch ID

iCloud 동기화

기기에서 iCloud를 켜두면, Writeblock은 Apple의 CloudKit 서비스를 이용해 일기를 여러 Apple 기기에 동기화할 수 있습니다. 이 동기화는,

  • 사용자가 제어합니다 — 설정에서 언제든 iCloud 동기화를 켜고 끌 수 있습니다(기본값 켜짐)
  • 전 과정을 Apple이 관리합니다 — Writeblock은 자체 서버를 운영하지 않습니다
  • Apple의 개인정보처리방침을 따릅니다 — iCloud 데이터는 Apple의 암호화와 개인정보 보호 정책으로 보호됩니다
  • 삭제할 수 있습니다 — 기기의 설정에서 iCloud에 동기화된 데이터를 모두 삭제할 수 있습니다

구매

Writeblock은 단건 인앱 구매가 있는 무료 앱이며, 결제는 전적으로 Apple이 App Store를 통해 처리합니다. Writeblock은 어떤 결제 정보도 받거나 저장하지 않습니다. 모든 구매 거래에는 Apple의 개인정보처리방침이 적용됩니다.

Screen Time API (FamilyControls)

Writeblock은 선택한 앱을 차단하기 위해 Apple의 Screen Time API(FamilyControls)를 사용합니다. 이 API는 불투명 토큰을 사용하므로 — Writeblock은 당신이 선택한 앱의 이름·번들 식별자나 그 밖의 어떤 정보도 볼 수 없습니다. 선택 내용은 Apple이 정한 방식으로 기기 안에 저장됩니다.

아동 개인정보 보호

Writeblock은 만 13세 미만 아동을 대상으로 하지 않습니다. 앱은 아동의 어떤 정보도 의도적으로 수집하지 않습니다.

당신의 권리 (GDPR / CCPA)

Writeblock은 신원을 식별할 수 있는 정보를 수집하지 않으며 — 당신과 연결할 수 없는 익명 사용 통계만 다룹니다 — 따라서 접근·정정·삭제·이동할 개인정보 자체가 없습니다. 일기와 개인 데이터는 기기 안에 저장되며 전적으로 당신의 통제 아래 있습니다.

  • 접근 — 앱을 열면 언제든 당신의 데이터를 볼 수 있습니다
  • 내보내기 — 설정의 내보내기 기능으로 일기를 Markdown으로 내려받을 수 있습니다
  • 삭제 — 설정의 "모든 데이터 삭제"로 모든 로컬 데이터를 영구히 지울 수 있습니다
  • 이동성 — 언제든 표준 형식으로 데이터를 내보낼 수 있습니다. 이 기능은 항상 무료입니다

데이터 보안

  • 일기는 기기의 Core Data에 저장됩니다
  • 앱 잠금 PIN은 iOS 키체인에 저장되며, 기기가 잠금 해제된 상태에서 이 기기에서만 접근할 수 있습니다
  • iCloud 데이터(켜둔 경우)는 Apple이 암호화합니다
  • 당사로 가는 데이터는 익명 사용 통계뿐입니다(PostHog, EU 호스팅). 일기 내용은 당사로 전송되지 않으며, iCloud 동기화를 켜면 당신 본인의 Apple iCloud 계정으로만 동기화됩니다(당사는 볼 수 없음)

방침 변경

이 개인정보처리방침을 업데이트하면, 갱신된 "최종 업데이트" 날짜와 함께 수정본을 이곳에 게시합니다. 익명의 비식별 사용 데이터만 수집하므로, 변경이 당신의 개인정보에 영향을 줄 가능성은 거의 없습니다.
